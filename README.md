# Proyecto Tienda de celulares

Este proyecto simula una tienda de compra de celulares donde en la primera pagina se pueden
ver todos los celulares disponibles ademas se pueden ordenar de mayor a menor y de menor a 
mayor precio y se pueden filtrar por marca.

Al dar click en VER PRODUCTO se redireccionara a una segunda pagina donde se mostraran los detalles 
del producto seleccionado con opcion de elegir color, almacenamiento y cantidad, al dar click en 
AGREGAR AL CARRITO el producto seleccionado y configurado se agregara al carrito que se podra ver 
en la primera pagina y tambien en la segunda.

## Realizado con:
- HTML
- CSS
- JAVASCRIPT
